# Talent Search Africa

Talent Search Africa is a web platform that allows African youth to showcase their skills, build digital CVs, upload portfolios, verify certificates, and connect with local and global recruiters.

---

## Features

- User registration and login (email/password, Google)  
- Role-based dashboards: Youth, Recruiter, Admin  
- Digital CV / Profile Builder  
- Portfolio upload (images, videos, PDFs)  
- Certificate verification workflow  
- Recruiter job posting and Applicant Tracking System (ATS)  
- AI Career Chatbot (English & Swahili)  
- Real-time messaging  
- Ratings & reviews  
- Notifications  
- Bilingual interface (English/Swahili)

---

## Tech Stack

- **Frontend:** React + TypeScript  
- **Styling:** Tailwind CSS + custom CSS  
- **Backend:** Firebase (Authentication, Firestore, Storage)  
- **Build Tool:** Vite  
- **Languages & Formats:** TypeScript (.ts/.tsx), JavaScript (ES6+), CSS, JSON
